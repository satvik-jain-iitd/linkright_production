Phase 01 Enter

First touch — land, understand the promise, create an account. The user decides in 30 seconds whether LinkRight is worth their email.

Landing→ Pricing→ Auth
Phase 02 Build your profile

The memory layer. Instead of a dev-tool wall, we ask one thing — drop your resume — and surface what we extracted. The user edits, approves, sets preferences. This is the foundation every downstream feature retrieves from.

Upload resume→ Review highlights→ Edit a highlight→ Preferences
Phase 03 Act — build & apply

The first magic moment. User finds a role, watches Sync plan the layout, then watches bullets get written live from their own memory layer. Ends with a pixel-perfect, ATS-safe PDF.

Find roles→ Layout plan→ Live writing→ Review & ship
Phase 04 Grow — the daily loop

The returning user. Dashboard becomes the morning ritual: today's matches, interview prep, the Kanban, Broadcast. The diary feeds the memory layer. This is where LinkRight compounds.

Dashboard→ Interview prep→ Applications→ Broadcast→ Diary→ Profile + notifs
